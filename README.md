A skill for Amazon Alexa, which allows you to play any episode of the Bit Socket Podcast.

Example utterances:

* Alexa, ask Bit Socket to play episode two
* Alexa, ask Bit Socket to list episodes
* Alexa, open Bit Socket
  * latest
  * Alexa, previous
  * Alexa, next
  * Alexa, pause
  * Alexa, resume
